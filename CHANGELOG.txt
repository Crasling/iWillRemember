================================================================================
 iWillRemember - Changelog
================================================================================

v0.16.2 (Release)
--------------------------------------------------------------------------------

 Options Panel - Sidebar Navigation
  - Replaced horizontal tabs with a vertical sidebar menu
  - Fixed empty options menu bug
  - All settings tabs now scroll when content overflows

 Customizable Button Labels
  - Rating names (Respected, Liked, Disliked, Hated) can now be renamed
  - Custom names show everywhere: buttons, tooltips, warnings, database
  - Added edit boxes and a reset button in the new Customize page
  - Custom names are per-user only and do not affect sync

 Custom Icon Selection
  - New "Customize" page in the settings sidebar for icons and labels
  - Pick custom icons for each rating type from a visual grid
  - Custom icons show on buttons, tooltips, and the database list
  - Chat icons stay as the default circular icons for a clean look
  - 22 curated WoW icons included (positive and negative themes)
  - Manual icon path input for any WoW icon with help text
  - Each rating has Change and Reset buttons with a live preview

 Localization
  - Added Russian translations for all new settings and UI text

v0.16.1 (Release)
--------------------------------------------------------------------------------

 Options Panel
  - Replaced custom options panel with AceConfig-3.0
  - Added four tabs: General, Sync, Backup, About
  - General: Target frame toggle, chat icons toggle, sound warnings toggle
  - Sync: Data sharing toggle, friend list dropdown for whitelist management
  - Backup: Hourly backup toggle, backup info display, database restore
  - About: Debug mode toggle, addon info, Discord link, credits

 Options Panel (Custom Standalone Frame)
  - Replaced AceConfig with a fully custom standalone settings frame
  - Styled section headers with tinted backgrounds and orange accents
  - Added description text under every toggle
  - Added "Minimap Settings" section with show/hide toggle
  - Added "Database Statistics" in the Backup tab
  - Added "Reset Settings to Defaults" with confirmation dialog
  - Reorganized About tab with addon icon, Discord link, translations, debug info
  - Added Blizzard Settings stub panel for discoverability
  - Added /iwr slash command (settings, db, menu)
  - Settings frame hides automatically on combat entry
  - All dynamic content refreshes when the panel opens

 Minimap Button
  - Right-click now opens the settings panel

 Target Frame (Cross-Version Compatibility)
  - Fixed target frame not updating on TBC / WotLK / MoP
  - Added fallback event listener for clients without the default update hook
  - Added overlay frame fallback for clients missing the target frame texture
  - Added safety checks to prevent crashes before the addon panel is opened
  - Target frame now resets properly when clearing a note or changing target

 Chat Icons
  - Fixed clickable chat icons not working on TBC / WotLK / MoP
  - Chat icon click hook now works reliably across all client versions
  - Fixed errors on system messages and channel notices

 Group Warnings
  - Fixed group scanning error when checking the 5th party member
  - Fixed group check running repeatedly for players not in the database

 Localization
  - Added ~15 locale strings for settings descriptions and UI elements
